参数说明

server: 数据源
     netease 网易云音乐(默认)
     tencent QQ音乐

type: 类型
     name 歌曲名
     artist 歌手
     url 链接
     cover 封面
     lrc 歌词
     single 获取以上所有信息(单曲)
     playlist 获取以上所有信息(歌单)

id: 单曲ID或歌单ID

GitHub：meting-api，此API基于 Meting 构建。

例如：https://tool.wxwz.xyz/tools/meting/?type=url&id=416892104
https://tool.wxwz.xyz/tools/meting/?type=single&id=591321
https://tool.wxwz.xyz/tools/meting/?type=playlist&id=6952531030
